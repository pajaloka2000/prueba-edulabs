Intalacion y configuracion:
    En consola:
    1) instalar las dependencias de node: npm install
    2) instalar las dependencias de composer: composer update

    Importar la base de datos:
    1) el archivo se encuentra en la carpeta sql
    2) crear una base de datos con el mismo nombre de la exportada: prueba-edulabs
    3) hacer la importacion en su gestor preferido, video de referencia: https://www.youtube.com/watch?v=jokDZXRwJ4o

    Para correr la aplicacion si se tiene instalados los servicios de php:
    1) ingresar a la carpeta public: cd public
    2) iniciar el servidor: php -S localhost:3000
    3) ir a la ruta: localhost:3000/login
